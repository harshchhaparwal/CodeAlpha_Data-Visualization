# CodeAlpha — Task 3 : Data Visualization

## 📄 Project Title
Data Visualization on Books Dataset (Task 3)

## 📘 Description
This project (Task 3) creates clear, insightful visualizations from the `books.csv` dataset produced in Task 1.  
It generates static PNG charts, an HTML report embedding the charts, and an optional interactive Plotly-based HTML dashboard.
